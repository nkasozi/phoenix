## Prefect Setup for Phoenix

Phoenix uses Prefect as its orchestration tool, ensuring optimal and reliable execution of workflows.

Prefect setup typically involves configuration for both local and production clusters:

- **Local Cluster**: Refer to [`development_getting_started.md`](development_getting_started.md) for detailed instructions.
- **Production Cluster**: You can deploy either a self-hosted or cloud-based Prefect instance.
  Review `charts/main/README.md` and `charts/main/values.yaml` for setup options and configuration
  details.
